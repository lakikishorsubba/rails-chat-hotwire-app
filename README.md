# Rails Project

## Overview
This is a Ruby on Rails application built as a learning and experimental project. It demonstrates the implementation of several core Rails features including real-time communication, email configuration, file attachments, and modern frontend enhancements using Hotwire and Action Text.

## Features

- CRUD functionality using Rails scaffold
- Email sending with SMTP configuration
- WebSocket integration using Action Cable
- Real-time chat system
- Chat history support
- File and image uploads using Active Storage
- Multiple file attachments support
- Rich text content using Action Text
- Navigation system for better UI flow
- Hotwire integration for modern reactive UI updates
- Basic backend setup and configuration improvements

## Tech Stack

- Ruby on Rails (API / Full-stack)
- PostgreSQL (Database)
- Action Cable (WebSockets)
- Hotwire (Turbo + Stimulus)
- Action Text (Rich text editor)
- Active Storage (File uploads)
- SMTP (Email service)

## Purpose

This project was built to explore and practice key Rails concepts such as real-time features, email systems, file handling, and modern Rails frontend tools like Hotwire.
